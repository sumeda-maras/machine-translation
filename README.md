# Transformers: Machine Translation
Developed a Transformer-based machine translation model using Natural Language Processing (NLP) to translate text between English and German. The project covers the complete workflow from data preprocessing and model training to evaluation and interactive translation.

## Dataset
The model is trained and evaluated using the **Multi30K** dataset, a multilingual image-caption dataset containing parallel English–German sentence pairs.

## Tech
- Python
- PyTorch
- Hugging Face Tokenizers
- TorchMetrics (SacreBLEU)
- Jupyter Notebook
